Browser Dialog Methods - BOM

Browser provides three main dialog methods: alert(), confirm(), and prompt(). These are simple ways to interact with users.

Important: These dialogs block the browser and may be annoying to users. Modern web development prefers custom modal dialogs.

Alert Dialog - Information Messages

alert() shows a message with an OK button.

Confirm Dialog - Yes/No Questions

confirm() shows a message with OK and Cancel buttons. Returns true/false.

Prompt Dialog - User Input

prompt() shows a message with an input field and OK/Cancel buttons.
