Browser Object Model (BOM)

What is BOM?

Browser Object Model (BOM) represents the browser window and provides objects to interact with the browser itself (not the document content).

Unlike DOM which deals with the document content, BOM deals with the browser window, location, history, etc.

Main BOM Objects

  • window - The global object, represents browser window
  • location - Information about current URL
  • history - Browser history
  • navigator - Information about the browser
  • screen - Information about user's screen

Key Points to Remember

  • BOM is not standardized (unlike DOM)
  • window is the global object in browsers
  • All global variables become properties of window
  • BOM objects help interact with browser features
