DOM Basics - Beginner Level

Welcome to JavaScript DOM (Document Object Model) basics. The DOM represents the structure of HTML documents as objects that JavaScript can manipulate.

1. What is the DOM?

The DOM is a programming interface for HTML documents. It represents the page so that programs can change the document structure, style, and content.

// The document object represents the entire HTML page console.log(document); console.log(document.title); console.log(document.URL);

2. Selecting Elements by ID

Use document.getElementById() to select an element by its ID attribute.

// Syntax: document.getElementById('elementId') let element = document.getElementById('myText');

This is a paragraph with ID 'myText'

3. Selecting Elements by Class Name

Use document.getElementsByClassName() to select elements by their class name.

// Returns a collection of elements let elements = document.getElementsByClassName('demo-class');

4. Selecting Elements by Tag Name

Use document.getElementsByTagName() to select elements by their tag name.

// Select all paragraph tags let paragraphs = document.getElementsByTagName('p');
This is a span element Another span element Third span element

5. Query Selectors (Modern Approach)

Use document.querySelector() for the first match and document.querySelectorAll() for all matches.

// CSS selector syntax let firstPara = document.querySelector('p'); // First paragraph let allParas = document.querySelectorAll('p'); // All paragraphs let specificClass = document.querySelector('.my-class'); // First element with class let specificId = document.querySelector('#my-id'); // Element with ID

6. Changing Element Content

Use innerHTML, textContent, and innerText to change element content.

element.innerHTML = '<strong>Bold text</strong>'; // Can include HTML element.textContent = 'Plain text only'; // Text only, no HTML element.innerText = 'Visible text only'; // Respects styling

7. Changing Element Attributes

Use getAttribute(), setAttribute(), and removeAttribute() to work with attributes.
